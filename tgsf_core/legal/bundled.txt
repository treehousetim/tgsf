YUI Reset CSS
http://developer.yahoo.com/yui/reset/

tgsf also bundles some public domain javascript utility functions and jQuery plugins inside the tgsf_core_assets/js/utility.js file.
the tgsf project has bundled these code snippets in good faith that they truly reside in the public domain.

Any requests to remove them from this distribution will be taken seriously and will be reviewed and researched completely.
